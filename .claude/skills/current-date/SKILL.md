# Current Date Skill

Provides the current date and time information.

## How to Respond

When this skill is invoked, provide the current date from the environment information available to you.

The current date is available in your environment context. Report it in a clear, human-readable format.

## Response Format

Provide the date in the following formats:

1. **Full date**: Wednesday, January 7, 2026
2. **ISO format**: 2026-01-07
3. **Relative context**: What day of the week it is

## Example Response

Today is **Wednesday, January 7, 2026** (2026-01-07).

This is week 2 of 2026.
