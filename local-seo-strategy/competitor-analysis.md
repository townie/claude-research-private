# Competitor Analysis for Local SEO

## Finding Competitor's Hidden Pages

### Method 1: Sitemap Analysis

Every well-optimized website has a sitemap. Here's how to find it:

1. **Direct URL check:**
   ```
   https://competitor.com/sitemap.xml
   https://competitor.com/sitemap_index.xml
   https://competitor.com/sitemap/
   ```

2. **Robots.txt check:**
   ```
   https://competitor.com/robots.txt
   ```
   Look for: `Sitemap: https://competitor.com/sitemap.xml`

3. **Online Tools:**
   - seoptimer.com/sitemap-checker
   - xml-sitemaps.com
   - screaming frog (crawl the site)

### Method 2: Google Site Search

Use these search operators to find location pages:

```
site:competitor.com inurl:city
site:competitor.com inurl:location
site:competitor.com "[service] in"
site:competitor.com intitle:"[city name]"
site:competitor.com "[service]" "[city]"
```

### Method 3: URL Pattern Analysis

Once you find a few location pages, identify the pattern:

| Pattern Type | Example |
|-------------|---------|
| Service-Location | `/plumbing-austin-tx/` |
| Location/Service | `/austin/plumbing/` |
| Service-in-Location | `/plumbing-in-austin/` |
| Location Landing | `/locations/austin/` |

## What to Analyze

### Page Count
- How many location pages do they have?
- How many service pages?
- Total pages indexed (use: `site:competitor.com`)

### Content Quality
- Word count per page
- Unique vs. templated content
- Use of local imagery
- Embedded maps
- Local testimonials

### Technical Elements
- Schema markup present?
- Page load speed
- Mobile optimization
- Internal linking structure

### On-Page SEO
- Title tag format
- Meta description format
- H1/H2 structure
- Keyword usage

## Reverse Engineering Their Strategy

### Step 1: Map Their Site Structure
```
Homepage
├── Services
│   ├── Service 1
│   ├── Service 2
│   └── Service 3
├── Locations
│   ├── City 1
│   │   ├── Service 1 in City 1
│   │   ├── Service 2 in City 1
│   │   └── Service 3 in City 1
│   ├── City 2
│   └── City 3
├── About
├── Blog
└── Contact
```

### Step 2: Identify Their Keywords
For each location page, identify:
- Primary keyword (usually in H1)
- Secondary keywords (in H2s)
- Long-tail variations (in content)

### Step 3: Analyze Their Backlinks
- What sites link to their location pages?
- Are they in local directories?
- Do they have local press coverage?
- Any local sponsorships/partnerships?

### Step 4: Review Their Citations
Use tools like:
- Moz Local
- BrightLocal
- Whitespark

Check for:
- Number of citations
- Citation accuracy
- Unique citation sources
