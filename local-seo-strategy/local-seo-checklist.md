# Local SEO Checklist

## Google Business Profile

- [ ] Claim and verify your Google Business Profile
- [ ] Complete all business information sections
- [ ] Add accurate business hours (including special hours)
- [ ] Select appropriate primary and secondary categories
- [ ] Write a compelling business description with keywords
- [ ] Add high-quality photos (exterior, interior, team, products/services)
- [ ] Enable messaging
- [ ] Add products/services with descriptions
- [ ] Set up booking/appointment links if applicable
- [ ] Respond to all reviews (positive and negative)
- [ ] Post updates weekly (offers, events, news)
- [ ] Add Q&A with common questions pre-answered

## Website Optimization

### Technical SEO
- [ ] Mobile-responsive design
- [ ] Fast page load speed (<3 seconds)
- [ ] SSL certificate installed (HTTPS)
- [ ] XML sitemap created and submitted
- [ ] Robots.txt properly configured
- [ ] Schema markup implemented (LocalBusiness)
- [ ] NAP consistent in footer/contact page

### On-Page SEO
- [ ] Title tags include location + service
- [ ] Meta descriptions are unique and compelling
- [ ] H1 tags include primary keyword
- [ ] Image alt text includes location where relevant
- [ ] Internal linking structure in place
- [ ] Contact information on every page

### Location Pages
- [ ] Unique page for each service area
- [ ] Minimum 500 words of unique content per page
- [ ] Local landmarks/neighborhoods mentioned
- [ ] Embedded Google Map
- [ ] Location-specific testimonials
- [ ] Local images (not stock photos)
- [ ] Clear call-to-action

## Citations & Directories

### Major Directories
- [ ] Google Business Profile
- [ ] Bing Places

- [ ] Yelp

- [ ] Foursquare

### Industry-Specific Directories
- [ ] Identify top 10 directories for your industry
- [ ] Create/claim listings on each
- [ ] Ensure NAP consistency
- [ ] Add complete business information

### Local Directories
- [ ] Chamber of Commerce
- [ ] Local business associations
- [ ] City/town business directories
- [ ] Regional directories

## NAP Consistency

- [ ] Audit all existing citations
- [ ] Document your official NAP format
- [ ] Update inconsistent listings
- [ ] Monitor for new/incorrect listings
- [ ] Use same format everywhere:
  ```
  Business Name
  123 Main Street, Suite 100
  City, State 12345
  (555) 123-4567
  ```

## Reviews Strategy

- [ ] Create a review generation process
- [ ] Send follow-up emails/texts requesting reviews
- [ ] Make it easy with direct review links
- [ ] Respond to all reviews within 24-48 hours
- [ ] Address negative reviews professionally
- [ ] Never buy fake reviews
- [ ] Display reviews on website

## Local Link Building

- [ ] Sponsor local events/charities
- [ ] Join local business associations
- [ ] Partner with complementary businesses
- [ ] Get featured in local news/blogs
- [ ] Create local resource content
- [ ] Offer expert quotes to journalists (HARO)
- [ ] Host/sponsor community events
